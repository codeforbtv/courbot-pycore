FAQ

1. How do I find my JP13 Court Case Number?
If you've been served papers, you should have an ID that starts with "JP13", a dash, a two-digit year, another dash, and six digits.
2. What if I haven't been served papers?
You can search CourtConnect for your name. Tip: use Case Type "61 - JP LANDLORD TENANT"
3. How is CourtBot DE associated with the state court system?
CourtBot Delaware is not affiliated with the Delaware Court systems and is a volunteer-provided pro-bona service created and maintained by Open Data Delaware, a Code for America Brigdae.
4. How will my information be secured?
First, we only capture data items relevant to your case schedule so we can send you a reminder. Once your case is closed, we will irreversably obfuscate your cell number using a hash to prohibit using it further.
5. How do I unsubscribe?
Text "UNSUB" to the 302-555-0123 from the cell phone you want to unsubscribe. If you've previously subscribed, you will receive confirmation of being removed. You can also text OPTOUT to prevent your number from being added again in the future.
